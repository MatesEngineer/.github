### 株式会社メイツとは？

Web プラットフォームのアプリケーションを主軸とした日本の EdTech 企業です．\
学習塾向け ICT 教材 [aim@](https://mates-app.jp/aim/) や，塾管理システム **reco** を始めとしたアプリケーションの開発・運用や研究開発，**学習塾の運営** を行なっています．\
ほぼ全てのシステムで Vue.js / TypeScript / AWS を使用しており、Vue や AWS はもちろん，様々なコミュニティでの活動や OSS 活動をしているメンバーも多数在籍しており，チーム全体で OSS への参加が前向きな組織です．

詳細な情報や，ブログはこちら: [https://eng.mates.education](https://eng.mates.education)
